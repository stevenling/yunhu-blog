# 《美学原理》第一讲 美学 为什么与是什么

# 一、美学为何：美学的学科特点

# 1.1 对审美现象的反思

去电影院看电影是一种审美活动，那么一部好电影的评判标准是什么呢？

第一种观点：好电影就是能让我产生生理反应，比如喜剧让我发笑，悲剧让我流泪。

第二种观点：生理反应是人类的一种低级反应，审美是一种高级反应，不能拿生理反应作为一个重要的评价标准。

我比较认同第二种观点，很多烂俗的电影也会让我笑，笑完就感觉空虚，而且没有再看一次的欲望，因此我认为没有艺术价值不是好电影。

美学不是在评价这部电影是好或者坏，而是研究当你在评价一部电影好坏的时候参照的是什么标准，背后是一种什么样的思维方式。

三观不正能否成为评价一个艺术作品好坏的标准？

艺术作品是否要遵守道德约束？

第一种观点：艺术世界也不是法外之地，依然要遵守道德，所有的艺术作品都是对现实生活的反应，现实生活中要遵循道德，那么艺术作品也要遵循。

第二种观点：艺术也要遵循道德，不是因为道德本身，而是因为艺术追求美，而美与道德约等同。

第三种观点：艺术作品可以呈现不道德的东西，但是要对这些进行批判，直白点就是坏人不能有好下场。

第四种观点：艺术并不是追求美的，而是追求人性可能性的一种探索，那么就不能预先设定道德立场，过分强调道德的约束会伤害作品的艺术价值。

第五种观点：艺术创作是自由的，但是艺术传播不是自由的，因此不是要对艺术作品进行源头上的限制，而是要对特定艺术作品进行范围的限定，比如分级制度。

# 1.2 对审美规律的探寻

人的外貌好看的评价标准？

头包脸，三庭五眼比较均衡，下颌角 120°，鼻面角 30°，鼻唇角 95°，Ricketts 审美线，眉间到鼻梁的大双 C 线。

美是否有规律？

如果美有规律，会不会导致千篇一律，审美疲劳。美有规律，规律意味着统一，统一意味着求同，那就会否定审美具有多样性的可能。

如果美没有规律，萝卜青菜各有所爱，你认为刘亦菲好看，我就认为她不好看，那完了，美学学科崩塌了，大家凭自己喜好就好了，不用探讨了。

美学不是学以致用，是学以致智的。学美学课并不会变美，很多美学家都是普通颜值。

建筑的美

从具象到抽象，人会感觉抽象比较美。那么为什么是这样的？

解释一：具象的事物是生活中常见的，而建筑是比较大的，因此会有一种比例的不协调，不舒适。建筑设计的精髓是把抽象的概念与具体的功能结合的巧妙。

解释二：审美的本质是无功利无利害的，太具象就显示出功利，让人感觉不美。

解释三：具象的让大多数人一看就知道，抽象得配合思考想一想，人会认为经过思考的比较高级。审美的背后是一种权力，一种趣味的间隔，一种权力的结构，一种带有阶级性的产物。

突然想到一个关于女生的，距离产生美，不熟悉的时候感觉好可爱，好干净，好美好。熟悉了发现她也是普通女孩，也世俗，也会有脚臭，腋毛等，认识从抽象到具象的一个祛魅的过程。看来女生只能留在想象里。

审美是否有高低，审美是否存在鄙视链？

我们这一代人很喜欢吐槽父母的审美，天然的觉得自己的审美是比较高级的，那么这种傲慢是从哪里来的？是因为我比父母那一辈获得了更多的知识和文化吗？

审美的背后可能是一种文化资本在支撑，审美从天然上可能就没有高低之分，很有可能是特定的社会背景，特定阶级铸造了特定的审美趣味。

# 1.3 对审美文化进行分析

当社会上形成了一种特定的审美类型、审美思潮的时候，那么就值得去探讨一下背后的社会与文化背景。

现在分析一下「赛博朋克」这个审美现象。

赛博朋克的视觉元素包括光怪陆离、霓虹灯、高楼大厦、地下贫民窟、浑浊空气、义体、义肢、高科技低生活。

赛博朋克的名称是由两个词构成，一个是科学控制论，另一个是朋克。朋克是起源于 1970 年的一个反文化潮流，核心就是反叛，叛逆。

赛博朋克的主要作品有《神经漫游者》、《银翼杀手》、《仿生人会梦见电子羊》。

赛博朋克世界中，人的记忆也是一种数据，当把记忆数据传输到仿生人里，那么仿生人就会认为自己是人。那么问题来了，我们如何证明自己不是仿生人，其实背后是对人的本质的探讨。

赛博朋克是一个反乌托邦的世界，科技越来越发达，富人的能力被强化，贫富差距拉大，穷人越来越穷而且无法反抗。

美学是对审美现象后的社会历史背景、意识形态等一系列问题加以更深入的追问。

# 二、美学何为：美学的学科性质

# 2.1 美学的学科定位

美学属于人文学科。

# 2.1.1 美学与自然科学

费希纳想通过科学实验对人的审美心理进行观察和定量的分析。

这样会产生一个问题，审美是具有历史性和社会性。现代人的审美与古人不同，不同文化背景的审美也不一样，单从自然科学角度无法完全解释。

目前人们好想认为「脸大」是不好看的，脸小是比较精致的。从进化论的角度解释脸大可能是因为咬合肌肉比较发达，那为什么有的人咬合肌肉比较发达？是因为这部分肌肉用劲比较多，那为什么这部分用了较多的劲，因为吃的比较多的是粗粮，脸小的人吃的细粮比较多。

人类历史上贫穷的人，底层的人吃的粗粮比较多，因此人类喜欢脸小的原因就是人类是嫌贫爱富的。

# 2.1.2 美学与社会科学

美学是从个体出发，社会科学从社会整体出发。

# 2.1.3 美学与人文科学

美学是人文科学的原因：

  1. 过程：主观介入，是一种价值判断。
  2. 对象：不具有实体性
  3. 结果：基于主观判断，无法验证。

# 2.2 美学的研究对象

# 2.2.1 美及美的规律（柏拉图）

美是什么？ 美是一朵美丽的鲜花，一匹骏马，一个漂亮的姑娘。然而这些都是美的特征，不是美本身。 美是什么？目前没有确切答案。

# 2.2.2 艺术

美学是艺术哲学。

黑格尔认为具有精神性的东西才具有美，因此自然没有精神性，不具有美的特征，他认为艺术具有精神性，艺术才具有美。

# 2.2.3 美感经验

  1. 审美经验：经验
  2. 审美心理：研究审美时的心理活动。

两派都认为纯粹客观的美不存在。

在人类出现以前，这个世界上存在美吗？

如果认为没有人也有美的存在，那么就认为美是一个绝对客观的产物，不依赖人。

如果认为没有人就没有美，那么就认为美是主观的产物。

# 2.2.4 人与现实的审美关系

美学研究的是一种主客体之间的一种关系。

认识世界的三种方式：

  1. 认识论：以科学研究的方式认识客观世界。
  2. 伦理学：通过主观经验感受道德问题。
  3. 美学：把握世界的第三种方式，以审美方式。

# 三、审美关系的四个维度

# 3.1 人与自身

心理学美学，格式塔心理学美学，视觉听觉美学，身体美学。

# 3.2 人与文化

艺术哲学

# 3.3 人与自然

环境美学

# 3.4 人与社会

社会美学，文化工业，政治上的美学，经济。
